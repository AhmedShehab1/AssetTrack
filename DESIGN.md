# AssetTrack Design System

## Visual Identity
*   **Brand Personality:** Professional, trustworthy, efficient, and modern.
*   **Vibe:** Sleek enterprise SaaS, minimalist, high legibility, generous whitespace.

## Color Palette
*   **Primary:** Deep Indigo (#3F51B5) or Electric Blue (#2196F3) for primary actions.
*   **Backgrounds:** Crisp White (#FFFFFF) for cards, Subtle Cool Gray (#F8F9FA) for page backgrounds.
*   **Neutral/Text:** Dark Slate (#212529) for headings, Slate Gray (#6C757D) for body text.
*   **Semantic Colors:**
    *   **Success (Available):** Emerald Green (#28A745)
    *   **Warning (Expiring Soon):** Amber/Gold (#FFC107)
    *   **Danger (Broken/Expired):** Crimson Red (#DC3545)
    *   **Info (Assigned):** Sky Blue (#17A2B8)

## Typography
*   **Font Family:** Inter or Roboto (Sans-serif).
*   **Headings:** Bold, high contrast for data points.
*   **Body:** Regular, optimized for readability in dense tables.
*   **Data:** Monospace for Serial Numbers (optional but helpful).

## Component Guidelines
*   **Data Tables:** Breathable rows, subtle borders, hover states, fixed headers.
*   **Status Badges:** Rounded pills with light background tints and saturated text/icons.
*   **Search/Filter Bars:** Integrated, clean inputs with clear iconography.
*   **Modals/Slide-outs:** Focused views with dimming overlays for depth.
*   **Cards:** Subtle box shadows (0 4px 6px -1px rgb(0 0 0 / 0.1)), rounded corners (8px-12px).
*   **Navigation:** Persistent sidebar with clear active states for Admin, Manager, and Developer roles.

## Layout
*   **Structure:** Single-page application (SPA) feel with a consistent sidebar or top-nav.
*   **Spacing:** Generous padding (16px-24px) to reduce cognitive load.
